Creates an INSERT command

Parameters:

  • pg_cur ‚Äì the psycopg cursor

  • table_schema ‚Äì the schema

  • table_name ‚Äì the name of the table

  • table_type ‚Äì the type of table, i.e. view or table

  • table_alias ‚Äì the alias of the table

  • remove_pkey ‚Äì if True, the primary is removed from the list

  • pkey ‚Äì can be manually specified.

  • coalesce_pkey_default ‚Äì if True, the following expression is used to insert the primary key: COALESCE( NEW.{pkey}, {default_value} )

  • skip_columns ‚Äì list of columns to be skipped

  • comment_skipped ‚Äì if True, skipped columns are written but commented, otherwise they are not written

  • remap_columns ‚Äì dictionary to remap columns

  • insert_values ‚Äì dictionary of expression to be used at insert

  • columns_on_top ‚Äì bring the columns to the front of the list

  • columns_at_end ‚Äì bring the columns to the end of the list

  • prefix ‚Äì add a prefix to the columns (do not applied to remapped columns)

  • returning ‚Äì returning command

  • add (indent) ‚Äì an indent in front

  • inner_defaults ‚Äì dictionary of other columns to default to in case the provided value is null (can be used instead of insert_values to make it easier to reuse other columns definitions)
